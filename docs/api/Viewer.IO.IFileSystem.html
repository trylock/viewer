Search(String, ISearchListener)

Search given path for files and subdirectories. OnDirectory(String) will be called for each directory, OnFile(String) will be called for each file.

If OnDirectory(String) returns Visit, the algorithm will search all its subdirectories. If OnDirectory(String) does not return Visit, its subdirectories and files won't be searched. If either OnDirectory(String) or OnFile(String) returns Abort, the search will stop immediately.

If path is a file, OnFile(String) will be called.

Declaration
void Search(string path, ISearchListener listener)
Parameters
Type Name Description
System.String path

Path to a file or directory
ISearchListener listener

Whenever a new directory is discovered, OnDirectory(String) is called. Whenever a new file is discovered, OnFile(String) is called.
